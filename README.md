## 简介

first-see 基于 SpringBoot 2.0+ 和一些主流开源组件，捣鼓的一个项目。

## 模块说明

```
first-see

├── wind-api         -- API服务
├── wind-common      -- 公共模块
├── wind-generator   -- 代码生成模块
├── wind-mapper      -- 公共的entity、mapper
└── wind-system      -- 后台管理系统
```
